## 1. Brand Overview

### 1.1 Brand Essence

Bloom Juice Room is a cold-pressed juice brand that embodies **feel-good energy through vibrant, plant-based nourishment**. The brand exists at the intersection of fun, health-conscious living and bold, playful design. Every touchpoint communicates freshness, vibrancy, and a commitment to real, unprocessed ingredients.

### 1.2 Brand Positioning

- **Category:** Cold-pressed juice / Plant-based beverages
- **Positioning:** Premium yet accessible wellness beverages made with real ingredients
- **Differentiators:** 100% plant-based, no added sugar, no preservatives, always fresh, fun bold branding
- **Target Audience:** Health-conscious consumers seeking convenient, nutritious, and delicious beverage options

### 1.3 Brand Promise

> *"Bloom Responsibly"* — Have fun, feel good, and make choices that are good for you and the planet.

### 1.4 Core Values

- **Fun:** Juice should make you feel good and put a smile on your face
- **Transparency:** Clear, honest communication about ingredients and benefits
- **Vibrancy:** Energy, freshness, and bold color in every touchpoint
- **Accessibility:** Premium quality that feels approachable and inclusive
- **Responsibility:** Plant-based, mindful choices that are good for you and the planet

## 3. Logo System

### 3.1 Primary Logo

The Bloom brand mark consists of two distinct typographic elements rendered in a rounded, geometric typeface. **No flower icon appears on any logo variation.**

**"Bloom" Wordmark**
- Custom rounded, geometric letterforms (based on Comfortaa)
- Thick, friendly shapes with soft curves
- Conveys approachability and playfulness
- Available in all five brand colors

**"JUICE ROOM" Descriptor**
- Rounded, geometric sans-serif typeface
- All caps, tracked out for clarity
- Provides structure and balance to the playful "Bloom" wordmark
- Positioned either beside (horizontal) or below (vertical) "Bloom"

### 3.2 Logo Variations

#### Horizontal Lockup
- **"Bloom"** wordmark with **"JUICE ROOM"** positioned to the side
- Primary usage for most horizontal applications
- Maintains clear hierarchy between wordmark and descriptor

#### Vertical Lockup
- **"Bloom"** wordmark with **"JUICE ROOM"** positioned below
- Used for square or tall format applications
- Creates a centered, stacked brand mark

#### Wordmark Only
- **"Bloom"** standalone — no descriptor, no flower
- Used when space is limited or brand is already established

**Note:** There is no "icon-only" logo variant. The 6-petal flower icon may be used as a decorative graphic element but is not part of the official logo lockups.

### 3.3 Color Variations

The logo is available in all five brand colors:

| Variation | Usage |
|-----------|-------|
| **True Black (#000000)** | Primary on light backgrounds |
| **True White (#FFFFFF)** | Primary on dark backgrounds |
| **Carrot (#FE6705)** | Accent applications, flavor-specific |
| **Kiwi (#C7D62A)** | Accent applications, flavor-specific |
| **Lychee (#FD7699)** | Accent applications, flavor-specific |

### 3.4 Clear Space & Minimum Size

- **Clear Space:** Minimum clear space equal to the height of the "B" in "Bloom" on all sides
- **Minimum Size:** 
  - Digital: 120px width for horizontal lockup
  - Print: 1.5 inches width for horizontal lockup
  - Icon only: 32px × 32px minimum

### 3.5 Logo Misuse

**Do NOT:**
- Stretch, skew, or distort the logo
- Change the colors outside approved variations
- Add effects (shadows, glows, outlines)
- Place on busy backgrounds without sufficient contrast
- Rotate the logo
- Separate "Bloom" and "JUICE ROOM" in the lockup
- Use outdated or unapproved versions

## 4. Color Palette

### 4.1 Core Brand Colors

Bloom's color system consists of five colors. These are the **only** approved brand colors. Colors can be combined in any combination as long as the result is readable.

| Color Name | HEX | RGB | CMYK | Usage |
|------------|-----|-----|------|-------|
| **True Black** | `#000000` | 0, 0, 0 | 0, 0, 0, 100 | Backgrounds, text, labels, logos |
| **True White** | `#FFFFFF` | 255, 255, 255 | 0, 0, 0, 0 | Text, labels, negative space, logos |
| **Carrot** | `#FE6705` | 254, 103, 5 | 0, 60, 98, 0 | Accent, logos, blobs, flowers, packaging |
| **Lychee** | `#FD7699` | 253, 118, 153 | 0, 53, 39, 0 | Accent, logos, blobs, flowers, packaging |
| **Kiwi** | `#C7D62A` | 199, 214, 42 | 20, 0, 90, 0 | Accent, logos, blobs, flowers, packaging |

**Note:** There are no secondary, accent, or extended palette colors. Everything uses only these five.

### 4.2 Color Usage Guidelines

#### Color Combinations
- **Black + White + One Accent Color:** Classic combination for most applications
- **Black + All Three Accent Colors:** Used for full-brand expressions, patterns, and environmental graphics
- **White + Accent Colors:** Used for light backgrounds, digital applications
- **Any combination is allowed** as long as readability is maintained

#### Accessibility
- Ensure minimum 4.5:1 contrast ratio for text
- Black on white or white on black provides maximum readability
- When using accent colors for text, ensure sufficient contrast with background
- Avoid using light accent colors (Kiwi, Lychee) for body text on white backgrounds

#### Color Psychology
- **Carrot:** Energy, warmth, vibrancy, creativity
- **Kiwi:** Health, freshness, nature, growth
- **Lychee:** Playfulness, sweetness, approachability, joy
- **True Black:** Sophistication, clarity, premium quality
- **True White:** Cleanliness, simplicity, freshness

## 5. Typography

### 5.1 Only Typeface: Comfortaa

**Comfortaa** is the **only** typeface allowed for the Bloom brand. No secondary or fallback typefaces are permitted.

#### Characteristics
- Geometric sans-serif with rounded terminals
- Friendly, approachable, modern
- Excellent readability at various sizes
- Available in multiple weights (via variable font)

#### Usage
- **Headlines:** Comfortaa Bold or SemiBold, **always all-caps**
- **Subheadings:** Comfortaa Medium, **all-caps**
- **Body Text:** Comfortaa Regular
- **Captions/Labels:** Comfortaa Light or Regular

#### Font Files
- `Comfortaa-Regular.ttf`
- `Comfortaa-Regular.woff`
- `Comfortaa-Regular.woff2`
- `Comfortaa-VariableFont_wght.ttf` (variable weight)

### 5.2 Typography Hierarchy

| Level | Typeface | Weight | Case | Size (Digital) | Size (Print) | Usage |
|-------|----------|--------|------|----------------|--------------|-------|
| **H1** | Comfortaa | Bold | ALL CAPS | 48px | 36pt | Main headlines |
| **H2** | Comfortaa | SemiBold | ALL CAPS | 36px | 28pt | Section headers |
| **H3** | Comfortaa | Medium | ALL CAPS | 28px | 22pt | Subsections |
| **H4** | Comfortaa | Medium | ALL CAPS | 22px | 18pt | Minor headers |
| **Body** | Comfortaa | Regular | Sentence case | 16px | 12pt | Body text |
| **Small** | Comfortaa | Regular | Sentence case | 14px | 10pt | Captions, labels |
| **Micro** | Comfortaa | Light | Sentence case | 12px | 9pt | Fine print, legal |

### 5.3 Typography Guidelines

- **Headlines:** Always all-caps, Comfortaa Bold or SemiBold
- **Line Height:** 1.5× for body text, 1.2× for headlines
- **Letter Spacing:** Slight tracking (+0.5px to +1px) for all-caps text
- **Alignment:** Left-aligned for body text, centered for headlines where appropriate
- **Hierarchy:** Use size, weight, and color to establish clear hierarchy
- **Avoid:** Using any typeface other than Comfortaa; using more than 3 type sizes on a single page/screen

## 7. Graphic Elements

### 7.1 Blob System

The blob system is a core graphic element of the Bloom brand, providing organic, dynamic visual interest.

#### Blob Characteristics
- Organic, flowing shapes
- Smooth curves, no sharp angles
- Varying sizes and proportions
- Can be solid color or multicolor
- Inspired by liquid, juice, and natural forms

#### Blob Variations

**Solid Color Blobs**
- Available in all brand colors: carrot, kiwi, lychee, true_black, true_white
- 20+ unique shapes (blob-01 through blob-20+)
- Used for backgrounds, accents, patterns
- Files: `blob-01-carrot.svg`, `blob-01-kiwi.svg`, etc.

**Multicolor Blobs**
- Combinations of two or three brand colors
- Create dynamic, layered visual interest
- 15+ unique combinations
- Files: `blob-01-carrot-kiwi.svg`, `blob-02-carrot-kiwi-lychee.svg`, etc.

**Side Blobs**
- Left and right variations for framing content
- Used for A-frame boards, banners, digital layouts
- Files: `side-left-carrot-kiwi-lychee.svg`, `side-right-carrot-kiwi-lychee.svg`, etc.

#### Blob Usage Guidelines
- Use blobs as background elements, accents, and layout framing devices
- Layer blobs for depth and dimension
- Combine blobs and flowers freely to form all sorts of backgrounds and layouts
- Maintain the five brand colors
- Ensure blobs don't interfere with readability
- Multicolor blobs can be used freely — no need for restraint
- Rotate and scale blobs as needed

### 7.2 Flower Graphics

Only the **primary 6-petal flower icons** from `assets/flowers/` should be used. No custom or alternate flower designs are permitted.

#### Usage
- **Pattern flowers:** Repeated flower motifs for backgrounds
- **Accent flowers:** Individual flowers for decoration
- **Large bullet points:** Flowers can be scaled up and used as decorative bullet points for lists
- **Neon flower:** Glowing flower for environmental graphics
- **Color variations:** Available in all five brand colors (filled and open/outline)
- **Combined with blobs:** Flowers and blobs are designed to be combined to form all sorts of backgrounds and layouts

### 7.3 Lines & Dividers

- **Horizontal lines:** Used to separate content sections
- **Rounded dividers:** Match brand's rounded aesthetic
- **Blob dividers:** Organic dividers using blob shapes
- **Usage:** Packaging, signage, digital layouts

## 11. Photography & Imagery

### 11.1 Photography Style

#### Characteristics
- **Bright and vibrant:** High key lighting, saturated colors
- **Fresh and clean:** Clean backgrounds, minimal clutter
- **Lifestyle-focused:** People enjoying juice, active lifestyles
- **Ingredient-focused:** Fresh fruits and vegetables, preparation process

- Enhance brand colors (Carrot, Kiwi, Lychee)
- Maintain natural juice colors
- Avoid overly filtered or artificial looks
- Ensure consistency across all imagery

- Clean, well-lit product shots
- Show juice color through transparent bottle
- Include fruit garnishes and ingredients
- Multiple angles: front, 3/4, top, lifestyle

#### Ingredient Photography
- Fresh, vibrant fruits and vegetables
- Show ingredients used in each juice
- Flat lays, overhead shots, detail shots

#### Settings
- Farmers markets
- Outdoor events

- Healthy lifestyle moments
- Social gatherings

#### Subjects
- Diverse, inclusive representation
- Active, healthy individuals
- Genuine moments, not overly posed

## 12. Layout & Composition

### 12.1 Grid System

#### Digital Grid
- **12-column grid** for web and digital applications
- **Gutter:** 24px
- **Margin:** 24px (mobile), 48px (desktop)
- **Baseline grid:** 8px

#### Print Grid
- **Modular grid** for print applications
- **Margins:** 0.5 inches minimum
- **Gutters:** Consistent spacing between elements

### 12.2 Spacing System

Use an 8px base unit for all spacing:

| Size | Pixels | Usage |
|------|--------|-------|
| **4px** | 4px | Tight spacing, icon padding |
| **8px** | 8px | Base unit, small gaps |
| **16px** | 16px | Standard spacing |
| **24px** | 24px | Medium spacing |
| **32px** | 32px | Large spacing |
| **48px** | 48px | Extra large spacing |
| **64px** | 64px | Section spacing |

### 12.3 Composition Principles

- **Hierarchy:** Establish clear visual hierarchy
- **Balance:** Balance elements for visual stability
- **Contrast:** Use contrast for emphasis and readability
- **Alignment:** Align elements to grid and each other
- **Proximity:** Group related elements together
- **White Space:** Use white space generously for breathing room

### 12.4 Layout Guidelines

- Maintain consistent margins and padding
- Use grid for alignment and structure
- Establish clear focal points
- Guide the eye through content
- Ensure readability at various sizes
- Test layouts on multiple devices/sizes
